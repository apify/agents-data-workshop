# Lesson 6: More Use Cases & Next Steps

**Goal:** Explore the MCP Configurator, try additional data sources through your agent, and see where to go from here.

**Time:** ~10 minutes

---

## 1. The MCP Configurator

The MCP Configurator shows which Apify Actors are exposed as tools your agent can call through MCP.

1. Go to [console.apify.com/mcp](https://console.apify.com/mcp)
2. Browse the list of available tools — each one maps to an Apify Actor
3. Toggle tools on/off to customize what your agent can access

## 2. Try more data sources

Ask your agent these prompts. Each uses a different Apify tool through MCP.

### Amazon reviews

> Scrape the latest reviews for the Kindle Paperwhite from Amazon and summarize the top complaints

### Google Maps

> Find the top-rated coffee shops in Berlin Mitte on Google Maps and show me their ratings and addresses

### Google Trends

> What are the trending search topics on Google Trends right now?

### TikTok

> Get TikTok posts from @nasa from the last week

Pick 2–3 that interest you. Each should return live data.

## 3. Explore the Apify Store

Go to [apify.com/store](https://apify.com/store). Every Actor can be called through MCP or the CLI. A few highlights:

| Platform | Actor |
|----------|-------|
| Google Maps | Business listings, reviews, contact info |
| Booking.com | Hotel prices, availability, ratings |
| TripAdvisor | Restaurant and attraction reviews |
| YouTube | Video metadata, comments, channel info |
| Facebook | Page posts, comments, events |
| LinkedIn | Company and job data |

## 4. You're done

You now have:

- Run Actors in the console, through MCP, and from the CLI
- Connected your AI agent to live web data via Apify's MCP server
- Installed Agent Skills that teach your agent how to use Apify
- Built and deployed your own Hacker News hiring scraper from a single prompt
